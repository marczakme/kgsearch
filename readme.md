# KG Entity Finder

Prosta wyszukiwarka encji Google Knowledge Graph zbudowana w Streamlit.

## Funkcje
- wyszukiwanie encji po nazwie
- filtrowanie po języku
- opcjonalne filtrowanie po typie encji
- eksport wyników do CSV
- podgląd surowej odpowiedzi JSON

## Uruchomienie lokalne

```bash
pip install -r requirements.txt
streamlit run app.py
